# Slides

DEMO of a simple web editor for Reveal.js using ACE Editor and Markdown, written in NodeJS.

^^^ Inspired by Hacker Slides

## Features
- Edit Mode
- Preview Mode
- (Todo) Save as Gist or Snippet

## Install
```
npm install
npm start
```

## Run
```
npm start
```

## Developer Mode
```bash
npm run dev
```
